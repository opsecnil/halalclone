# Halal Middleman Clone
### Overview
Complete fake Halal Middleman discord Bot that can be used to scam.

You can host it as a web service.

This project was made for educational purposes only.

### Features included:
- **Server switching**: find a victim already in the real Halal servers, invite them to your completely normal server, switch it to Halal and add them to a deal, they think they are in a real deal since they were only in the real servers (channels, server name and icon, owner and bot account going offline and hidden). The switch is fast and complete, all the other features are made to be compatible with the switching.
- **Owner always online**: the owner account "Peg" will stay online.
- **Staff Whitelisting**: owners can easely manage staff to help you.
- **Staff panel**: the staff will have access to a channel with buttons to manage the server.
- **Deal management**: the bot creates new wallets for every payment invoice, all the private keys are sent to the owner account "Peg", confirming the release of the money will kick the non-staff member from the server.
- **Auto-detection of transactions**: the bot auto-detects the transactions making it easy to use scamming as a seller.
- **Fake transactions**: the bot DMs the staff member of the deal with a panel to fake the transactions of the deal making it possible to put a fake transaction a fake transaction hash, generate a random one, or send money to the bot for real.
